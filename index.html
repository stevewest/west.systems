FlexAdmin

FlexAdmin allows for the administration of multiple physical machines with multiple virtual applications via a single control panel. Applications register menu items, forms and lists via a restful api that allows the admin application to dynamically build any kind of interface that the application needs. Included in this is a set of packages to allow for the remote administration of hardware functions such a network and power settings.

FlexRadius

FlexRadius is a full RADIUS management system build on top of FreeRADIUS. The functionality is split into separate applications to allow flexible deployment to suit clients' needs. The system provides an authenticated only method as well as an anonymous method of access that can be deployed as a single physical or virtual machine or spread over separate devices to allow for greater physical network access and scaling.

FlexReporter

FlexReporter is a report generation application designed to be a visual SQL query builder to allow data from any source to be regularly reported on, such as transactions, registered accounts or any data stored on your system. Reports can be exported in CSV for use in a 3rd party application or pdf, these can be downloaded as needed or automatically emailed to any number of people.
